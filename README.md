# VK SMM Assistant

> ⚠️ Проект в активной разработке. Не используйте в продакшене до версии 1.0.

Автоматизация ведения сообществ ВКонтакте: анализ конкурентов, генерация контента, публикация постов, аналитика и монетизация.
